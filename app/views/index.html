Crochet CAD

This site contains a couple of pattern generators for balls and donuts in single-crochet. The generators should be particularly useful for designing amigurumi.

I plan to add more generators in the future – I'm currently thinking about cones and cubes. If you have any suggestions for 3D primitives please let me know.

Note

The patterns assume that your stitch height is the same as your stitch width. This is not quite the case with single-crochet stitches, although the end-results seem to be okay. I am going to fix this at some point in the future!

All patterns are generated using US crochet stitch terms.
